# Code of Conduct

This is a small, open source project maintained by one person. Contributions are welcome — please keep interactions respectful and constructive.

## Expected behavior

- Be kind and considerate in issues, pull requests, and discussions
- Give and receive feedback graciously
- Assume good intent; ask before concluding bad faith

## Unacceptable behavior

- Insults, personal attacks, or derogatory comments
- Intimidation of any kind
- Publishing someone's private information without consent
- Any conduct that would be considered inappropriate in a professional setting

## Enforcement

This project is too small for a formal committee. If something feels off, open a GitHub Issue or reach out directly. I'll review it and respond fairly.
